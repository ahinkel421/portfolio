Hey! My name is Adam and I'm a Washington, DC-based full stack web developer.

Coming from a financial background, I've never had much room to use my creative muscle in my career. I started learning to code on Codecademy and put that creative muscle to work. It was somewhere between making web pages look awesome with CSS, and staying up all night trying to get those pages to function properly, that I really discovered my passion for web development. I went on to enroll in Thinkful's full-stack web development program, where I learned everything I know about this awesome craft. I'm very excited about what I've learned so far, but I'm even more excited about everything that I have yet to learn.
